условия аренды

Для физических лиц:
  • Гражданский паспорт
  • Водительское удостоверение
Способы оплаты для физических лиц:
  • Наличными при получении автомобиля
  • Банковским переводом
  • Банковской картой
Для юридических лиц:
  • Реквизиты организации
  • Доверенность на право осуществления действий от имени юредического лица (принимать и эксплуатировать ТС, подписывать договора и акты)
  • Паспорт и водительское удостоверение водителя
  • Копии должны быть заверены печатью организации и подписью генерального директора.
Способы оплаты для юридических лиц:
  • Только по безналичному расчету (работаем без НДС)
